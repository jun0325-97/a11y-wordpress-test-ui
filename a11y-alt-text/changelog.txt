*** A11Y - AI Alt Text & Accessibility Changelog ***

2026-04-22 - version 0.1.0
* 초기 프로토타입 릴리스
* 목업 API 모드 구현 — 실제 API 미연동 상태에서 UI/UX 테스트 가능, 더미 데이터 반환
* Mock Mode 배너 추가 — 테스트 모드임을 관리자 화면 전역에 명시적으로 표시
* 설정 페이지 UI 전면 개편 — 워드프레스 네이티브 관리자 스타일(wrap, form-table, notice) 준수
* 웰컴 패널 추가 — API 키 미연결 상태에서 3단계 초기 설정 가이드 표시
* 미디어 라이브러리 첨부파일 상세 화면 UI 개편 — alt 텍스트 + 상세 설명 필드 구조 정비
* alt 텍스트 + Long Description(aria-describedby) 이중 구조 지원 설계 완료
* 네트워크(멀티사이트) 설정 페이지 구조 지원
* 다국어(WPML, Polylang) 연동 지원
* WP-CLI 명령어 지원 (wp a11y generate)
